# Audit Studio Design System

A comprehensive, accessible design system built with Tailwind CSS, following W3C Design Tokens architecture and WCAG 2.2/2.1 AA standards.

## Navigation

- **[Foundations](/docs/foundations-overview--docs)** — Color, typography, spacing, and icons
- **[Components](/docs/components-button--docs)** — Reusable UI components

## Design Tokens

Our system follows a 4-level token architecture:

| Level | Purpose | Example |
|-------|---------|---------|
| 1 | Primitive | `#7C3AED` |
| 2 | Semantic | `--color-action-primary` |
| 3 | Component | `--button-primary-bg` |
| 4 | Theme | `--theme-action-primary` |

## Accessibility First

All components meet WCAG 2.2 Level AA requirements:
- Color contrast ratios of 4.5:1 for text
- Keyboard navigation support
- Screen reader compatible markup
- Focus indicators on all interactive elements
