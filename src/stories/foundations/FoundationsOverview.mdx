# Foundations

Foundations are the visual building blocks of the Audit Studio design system. They define the core design language through color, typography, spacing, and icons.

## What's Included

| Foundation | Description |
|------------|-------------|
| **[Colors](/docs/foundations-colors-overview--docs)** | Complete palette with primary, feedback, and semantic colors |
| **[Typography](/docs/foundations-typography-overview--docs)** | Font families, type scale, and text styles |
| **[Spacing](/docs/foundations-spacing--docs)** | Space scale, border radius, and shadows |
| **[Icons](/docs/foundations-icons-overview--docs)** | Icon library and usage guidelines |

## Token Architecture

All foundations are available as design tokens:

```
Level 1: Primitive → Base values (e.g., #7C3AED)
Level 2: Semantic → Contextual meaning (e.g., color-bg-brand)
Level 3: Component → Usage-specific (e.g., button-primary-bg)
Level 4: Theme → Mode overrides (light/dark)
```

## Accessibility

All foundations are tested for WCAG 2.2 AA compliance:
- Color contrast ratios
- Text sizes and readability
- Focus indicators
- Screen reader compatibility
